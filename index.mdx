Heart Disease Classification is a deep learning project that builds and evaluates a binary classification neural network to predict the presence of heart disease in patients. Using the well-known UCI Heart Disease dataset (1,025 patient records, 13 clinical features), it systematically explores different architectural choices — activation functions, optimizers, learning rates, and batch sizes — to understand their real-world impact on model performance.

## Getting started

<Steps>
  <Step title="Install dependencies">
    Create a virtual environment and install TensorFlow, scikit-learn, pandas, and Jupyter.
  </Step>
  <Step title="Download the dataset">
    The `heart.csv` file is included in the repository — no separate download needed.
  </Step>
  <Step title="Open the notebook">
    Run `jupyter notebook heart_disease_classification.ipynb` to launch the full experiment.
  </Step>
  <Step title="Explore results">
    Review the training curves, confusion matrix, and experiment tables generated by each cell.
  </Step>
</Steps>

<Note>
  This project was developed as a mid-term project for a Deep Learning course. The notebook is written primarily in Turkish but all code and results are universally interpretable.
</Note>
