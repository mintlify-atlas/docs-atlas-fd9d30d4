---
title: "Run the Heart Disease Classification Notebook Locally"
description: "Clone the repository, create a Python virtual environment, install all dependencies, and launch the Jupyter notebook to run every experiment."
---

This quickstart walks you through cloning the repository, setting up a Python virtual environment, installing all required dependencies, and launching the Jupyter notebook. By the end, you'll have the full experiment running locally and can start exploring the training cells.

<Steps>
  <Step title="Prerequisites">
    Make sure you have **Python 3.10 or later** installed on your system. You can check your version with:

    ```bash
    python3 --version
    ```

    You'll also need `pip` and `git` available on your `PATH`.
  </Step>

  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/metesahankurt/heart-disease-uci.git
    cd heart-disease-uci
    ```

    The repository includes `heart.csv` — the UCI Heart Disease dataset — so no separate data download is required.
  </Step>

  <Step title="Create a virtual environment">
    Create an isolated environment to avoid conflicts with other Python projects on your machine:

    ```bash
    python3 -m venv venv
    ```

    Then activate it:

    <CodeGroup>

    ```bash Linux / macOS
    source venv/bin/activate
    ```

    ```bash Windows
    venv\Scripts\activate
    ```

    </CodeGroup>

    Your terminal prompt should now show `(venv)` indicating the environment is active.
  </Step>

  <Step title="Install dependencies">
    Install all required packages with a single command:

    ```bash
    pip install tensorflow scikit-learn pandas numpy matplotlib seaborn jupyter
    ```

    This installs TensorFlow 2.x (which includes Keras), scikit-learn for preprocessing and metrics, pandas and numpy for data handling, matplotlib and seaborn for visualization, and Jupyter for running the notebook.
  </Step>

  <Step title="Launch the notebook">
    ```bash
    jupyter notebook heart_disease_classification.ipynb
    ```

    Jupyter will open in your browser. The notebook contains the full experiment — data loading, preprocessing, model definition, training runs, and results analysis.

    <Tip>
      Run cells in order from top to bottom. Each cell depends on variables and imports defined in earlier cells. Skipping ahead or re-running cells out of order can cause `NameError` or produce unexpected results.
    </Tip>
  </Step>
</Steps>

<Note>
  The notebook is written in Turkish, but all code, variable names, and output are universally readable. This documentation site provides English-language explanations for all experiments and results.
</Note>
